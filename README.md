Este proyecto consiste en el desarrollo de una página web utilizando HTML5 y CSS3. La página incluye una estructura semántica con etiquetas como header, main, article y footer, además de un menú de navegación, una imagen, listas y un formulario.

También se aplican estilos mediante un archivo CSS externo, utilizando variables CSS, selectores de clase, selectores de ID y efectos hover.
